# <img src="res/ImMusic.svg" width="32" style="vertical-align: -3px;"> An immediate mode music player using Dear ImGui (HEAVY WIP)


## About
This project is a remake of another music player I made, called [Qbar](https://github.com/Worm-Warrior/QBar) that is no longer being worked on.

The general idea is to make a fast and responsive music player that can play from both local files and over network from a Navidrome instance.

I wanted to make this because I was not very happy working with Qt for something so simple, so I switched to ImGui docking branch and it is exactly what I wanted.
